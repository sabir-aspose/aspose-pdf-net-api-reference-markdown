## OptimizationOptions.CompressAllContentStreams property {#optimizationoptionscompressallcontentstreams-property}

If set to `true`, all uncompressed page content streams will be compressed using the FlateDecode filter during [`OptimizeResources`](../../../aspose.pdf/document/optimizeresources/). Default is `false` to preserve backward compatibility.

public bool CompressAllContentStreams { get; set; }
