## SideBySideComparisonOptions.ExcludeAreas2 property {#sidebysidecomparisonoptionsexcludeareas2-property}

Get and set the exclude areas. Used for the second page or document in the comparison method. This option can be setted along with [`ExcludeTables`](../excludetables/). This option can’t be setted along with [`ComparisonArea2`](../comparisonarea2/) option.
