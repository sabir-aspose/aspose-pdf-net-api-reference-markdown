## AutoTaggingSettings.HeadingLevels property {#autotaggingsettingsheadinglevels-property}

Gets or sets the heading levels used for determining the structure of headings in a PDF document.

```csharp
public HeadingLevels HeadingLevels { get; set; }
```

## Remarks {#remarks}

The `HeadingLevels` property allows configuring the mapping of font sizes to heading levels. This is used during the auto-tagging process to identify and assign appropriate heading levels based on the font size of text elements in the document.
