## HtmlLoadOptions.CreateLogicalStructure property {#htmlloadoptionscreatelogicalstructure-property}

Gets or sets a value indicating whether to create a logical structure in the resulting PDF document.

```csharp
public bool CreateLogicalStructure { get; set; }
```

## Remarks {#remarks}

When set to `true`, the logical structure of the document is created, which can improve accessibility and make the document more suitable for screen readers and other assistive technologies.
