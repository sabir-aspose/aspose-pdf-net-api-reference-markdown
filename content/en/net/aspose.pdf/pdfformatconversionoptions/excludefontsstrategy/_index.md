## PdfFormatConversionOptions.ExcludeFontsStrategy property {#pdfformatconversionoptionsexcludefontsstrategy-property}

Strategy(ies) to exclude superfluous fonts and reduce document file size. This parameter has sense only when flag [`OptimizeFileSize`](../optimizefilesize/) is set to true. By default combination of strategies SubsetFonts and RemoveDuplicatedFonts is used.

public RemoveFontsStrategy ExcludeFontsStrategy { get; set; }
