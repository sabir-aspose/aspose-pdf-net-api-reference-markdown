## VectorStore.Metadata property {#vectorstoremetadata-property}

Gets or sets a set of 16 key-value pairs that can be attached to a vector store. This can be useful for storing additional information about the vector store in a structured format. Keys can be a maximum of 64 characters long and values can be a maximum of 512 characters long.
