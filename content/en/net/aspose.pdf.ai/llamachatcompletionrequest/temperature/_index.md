## LlamaChatCompletionRequest.Temperature property {#llamachatcompletionrequesttemperature-property}

Sets or gets the sampling temperature to use, between 0 and 2. Higher values like 0.8 will make the output more random, while lower values like 0.2 will make it more focused and deterministic. Default value is 1.
