## Signature.DefaultSignatureLength property {#signaturedefaultsignaturelength-property}

Gets or sets the default length for the signature data in bytes.

```csharp
public int DefaultSignatureLength { get; set; }
```

## Remarks {#remarks}

This is an estimation of the length of the signature in bytes. Used for signing via [`CustomSignHash`](../customsignhash/) if the [`AvoidEstimatingSignatureLength`](../avoidestimatingsignaturelength/) parameter is set. The default value is 3000.
