# Aspose.PDF for .NET API Reference Markdown

Generated Markdown snapshots of the official Aspose.PDF for
.NET online API reference.

## Authority

- Source: https://reference.aspose.com/pdf/net/
- Generated files: `content/`
- Crawl manifests: `manifests/`
- Audit reports: `audits/`

Files under `content/` are generated. Do not edit them manually. Fix the
converter or source mapping, then regenerate the affected platform.

## Current Platform

- Language: English
- Platform: net

## Monthly Updates

Each monthly SDK release should be published through a `refresh/YYYY-MM`
branch, reviewed in a pull request, merged into `main`, and tagged with
`pdf-YYYY-MM`.
